# blackjackgame
Juego de Blackjack clásico creado con JS, HTML5 y CSS3 (diseño futurista). La lógica usa un enfoque modular y gestión de estados para el puntaje (Ases dinámicos). Se aplican técnicas de CSS avanzado (3D, transform, perspective) para simular profundidad en las cartas y una UI responsiva. Incluye flujo de apuestas con modales.
